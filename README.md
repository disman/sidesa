## Tentang SIDESA

SIDESA merupakan sebuah Sistem Informasi Desa yang dikembangkan dengan tujuan untuk memudahkan Desa dalam mengelolah atau memanajemen data Desanya. Aplikasi ini dikembangkan menggunakan Laravel versi 9 sebagai backendnya, reactjs sebagai frontendnya dan tailwind css sebagai UInya.
